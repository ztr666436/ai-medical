## DeepSeek API 集成架构

```
health-data-service                    前端 (SSE 流式)
       │ Feign                               │
       ▼                                     ▼
┌──────────────────────────────────────────────────┐
│                AgentController                    │
│  /parse-health  /chat  /chat-stream  /advice     │
└─────────────────────┬────────────────────────────┘
                      │
┌─────────────────────▼────────────────────────────┐
│              AgentServiceImpl                     │
│  ├─ parseHealthData() → DeepSeekClient ──► API   │
│  ├─ chat() → 上下文管理 → DeepSeekClient ──► API  │
│  ├─ chatStream() → Flux<SSE> → DeepSeekClient    │
│  ├─ generateHealthAdvice() → DeepSeekClient ──►   │
│  └─ detectIntent() → DeepSeekClient ──► API       │
└──────────────────────────────────────────────────┘
                      │
┌─────────────────────▼────────────────────────────┐
│              DeepSeekClient (WebClient)            │
│  ├─ chatSync()     POST /v1/chat/completions      │
│  └─ chatStream()   POST /v1/chat/completions(SSE) │
└─────────────────────┬────────────────────────────┘
                      │ HTTPS
                      ▼
          ┌─────────────────────┐
          │  api.deepseek.com   │
          │  model: deepseek-chat│
          └─────────────────────┘
```

## 6 个 REST 接口

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 健康解析 | POST | `/api/agent/parse-health` | 供 health-data-service 调用 |
| 多轮对话 | POST | `/api/agent/chat` | 阿福风格对话 |
| **流式对话** | POST | `/api/agent/chat-stream` | **SSE 实时输出** |
| 健康建议 | POST | `/api/agent/health-advice` | 个性化建议 |
| 意图识别 | POST | `/api/agent/intent` | 路由分发 |
| 健康检查 | GET | `/api/agent/health` | 服务状态 |

## 4 套 System Prompt 模板

| Prompt | 用途 | 输出格式 |
|--------|------|---------|
| `HEALTH_PARSE` | 自然语言→结构化健康数据 | JSON |
| `CONVERSATION` | 蚂蚁阿福风格萌趣对话 | 文本 |
| `HEALTH_ADVICE` | 基于数据的健康建议 | 文本 |
| `INTENT` | 用户意图分类 | JSON |

## 降级策略

```
AI 调用成功 → 返回 DeepSeek 结果
       ↓ 失败
├─ parseHealth    → 返回空 records，health-data-service 规则引擎兜底
├─ chat           → "阿福正在休息，请稍后再和我聊天吧～"
├─ healthAdvice   → 返回通用鼓励语
├─ detectIntent   → 简单正则规则判断
└─ chatStream     → "流式服务暂时不可用"
```
